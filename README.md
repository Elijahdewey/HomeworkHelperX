# GradeFlow

Modern grade tracker for Oklahoma Christian School.

## Run locally

1. Install Node.js (v22+ recommended for built-in SQLite).
2. From this folder:
   - `npm install`
   - `npm run dev`
3. Open `http://localhost:3000`

The SQLite database file is created at `gradeflow.sqlite3` (set `GRADEFLOW_DB_PATH` to change it).
